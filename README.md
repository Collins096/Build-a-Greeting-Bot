# 🤖 Teacher Bot (JavaScript Project)

## 📌 Description

This is a simple JavaScript project that simulates a chatbot introducing itself.
It demonstrates basic programming concepts such as variables, string concatenation, and console output.

## 🚀 Features

* Introduces a bot with different names
* Displays location and favorite subject
* Uses variables and string manipulation

## 🛠️ Technologies Used

* JavaScript

## 📂 How to Run

1. Copy the code into a `.js` file (e.g., `script.js`)
2. Open your browser console or use Node.js
3. Run the file:

   ```bash
   node script.js
   ```

## 📸 Sample Output

```
Hi there!
I am excited to talk to you.
Allow me to introduce myself.
My name is teacherBot.
I live in the universe.
My nickname is professorBot.
I love my nickname but I wish people would call me awesomeTeacherBot.
My favorite subject is Computer Science.
Well, it was nice to talk to you. Have a nice day!
```
